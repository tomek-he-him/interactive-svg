&lt;drawing-board&gt;
=====================

**A navigatable SVG canvas.**  
**Made for web-based CAD apps.**




Installation
------------

*drawing-board* is available through NPM:

```sh
$ npm install drawing-board
```

– and through bower:

```sh
$ bower install drawing-board
```
